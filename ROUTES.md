# Site Routes

This document lists all the routes created for the Southaven Internal Medicine website and their corresponding original URLs.

## Generated Routes

| Route | Original URL | Description |
|-------|-------------|-------------|
| `/` | https://magdiwassef.wixsite.com/mysite | Homepage with hero, about, services, and contact sections |
| `/patient-portal` | https://magdiwassef.wixsite.com/mysite/patient-portal-at287 | Patient portal information and access |
| `/hours-location` | https://magdiwassef.wixsite.com/mysite/contact-us | Office hours, location, and directions |

## Route Mapping Notes

- The original `/mysite/contact-us` URL contained hours and location information, so it was mapped to `/hours-location` for better SEO and user experience
- The original `/mysite/meet-our-team` was mapped to `/meet-our-providers` to better reflect the medical nature of the practice
- The original `/mysite/patient-portal-at287` was simplified to `/patient-portal`
- All routes maintain the original content and functionality while providing cleaner, more professional URLs

## Navigation Structure

The main navigation includes:

All internal links have been properly mapped to the corresponding Astro routes, ensuring a seamless user experience throughout the site.
